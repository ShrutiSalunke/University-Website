# University Website UI

A responsive university/e-learning landing page built as a front-end UI practice project — covers a full multi-page layout with navigation, courses, campus, facilities, and testimonials sections.

## Features
- Responsive navbar with mobile toggle menu
- Sections for courses offered, global campuses, facilities, and student testimonials
- Star-rating testimonial cards
- Call-to-action and footer with social icons

## Tech Stack
HTML, CSS, JavaScript, Font Awesome
